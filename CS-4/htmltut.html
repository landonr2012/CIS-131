Basic HTML Tutorial

Getting Started

Hypertext Markup Language, or HTML, is the standardized format for structuring a web page. An HTML document consists solely of text and can be written in any text editor. Below are a few text editors for Mac and PC, but feel free to use what is comfortable for you.



For Windows:

Notepad++
Eclipse

For Mac:

BBEdit 11
Atom


Tags


To create a web page, you enter text that you want to display along with HTML code known as TAGS. Tags label the content of a web page such as titles, headings, paragraphs and tables. They always begin with an opening bracket (<) and end with a closing bracket (>). Every document starts with a DOCTYPE declaration, which provides information about the programming language used to write the page. This is known as a one-sided tag because it does not have a closing pair. Tag pairs are used to assign meaning to an element of a web page.




Elements


Elements are the building blocks of all web pages. They are specific components of a web page, such as a title, heading or paragraph. To use an element, you place an opening tag at the start of the element and a closing tag at the end. Closing tags are the same as its opening tag except it includes a slash (/) after the opening angle bracket.



Structure of a basic HTML Document


<!DOCTYPE html>

<html>

<head>

<title>

</head>

<body>


<h1> Heading </h1>

<p > Paragraph </h1>

</body>

</html>




Explanation of Example


•DOCTYPE html - defines the document to be HTML5
•html - element that is the root of any HTML page
•head - element that contains meta information about the web page
•title - element that specifies the title of the web page
•body - element that contains the visable web page content
•h1 - element that defines a heading. Can range from h1-h6 with h1 being the largest and h6 the smallest
•p - element that defines a paragraph



Adding Content to a Web Page


Adding text to a web page is simply finding the element you want to add content to and typing between the opening and closing tags.



Adding Text


<!DOCTYPE html>

<html>

<head>

<title> title of web page </title>

</head>

<body>


<h1> heading text </hl>

<p> paragraph text </p>

</body>

</html>





Using Comments


So far we have learned that HTML contains text that the browser displays, but there is another important type called comments. Comments are text elements that are not rendered by the browser. They are very helpful when creating large web sites or when collaborating with a team. Comments can be used to explain a particular section or to test whether an element is rendering properly. A common practice is to add a comment to the head section describing the content, filename, author and the data which it was last modified. In HTML, the only type of comment is the multi-line comment which begins with < !-- and ends with -->.



Adding Comments Example


<!DOCTYPE html >

<html>

<head>

<title> title of web page </title>

<!-- description of web page Filename: Author: Date: -->

</head>

<body>


<h1> heading text </h1>

<p> paragraph text </p>

</body>

</html>





Validating your HTML Document


Always validate your HTML code before uploading it! Validation is an automated process of comparing the code you've written against the HTML5 coding standards. It is a great way of identifying a problem that causes your page not to render like it should.
